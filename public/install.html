Installation Guide

This guide explains the basic setup process for Shadow Systems Discord bots. Each purchased bot may include its own extra setup notes depending on features.

1. What You Receive

After purchase, you will receive the files needed to run your Discord bot.

  • Full bot source code
  • Config or .env file
  • Setup instructions
  • Required package list
  • Basic Discord support

2. Install Node.js

Most Shadow Systems bots are built using Node.js and discord.js.

  • Download Node.js from the official website
  • Install the LTS version
  • Restart your computer if needed
  • Open Command Prompt or Terminal
  • Check install with node -v

3. Create Your Discord Bot

You will need a bot application from the Discord Developer Portal.

  • Create a new application
  • Add a bot to the application
  • Copy the bot token
  • Enable required intents
  • Invite the bot to your server

4. Edit Your Config

Your bot will include a config file or .env file for your private settings.

  • Add your bot token
  • Add your client ID
  • Add your guild/server ID
  • Add channel IDs
  • Add role IDs if needed

5. Install Packages

Open the bot folder in Command Prompt or Terminal and install dependencies.

  • Open the bot folder
  • Run npm install
  • Wait for packages to finish
  • Fix any missing package errors
  • Contact support if needed

6. Start The Bot

Once everything is configured, start the bot from your terminal.

  • Run node index.js
  • Confirm the bot comes online
  • Register slash commands if included
  • Test all commands
  • Check logs for errors

7. Hosting Options

Bots must stay running to remain online. You can run them locally or use hosting.

  • Run from your own computer
  • Use a VPS
  • Use bot hosting
  • Use game server hosting if supported
  • Ask Discord support for guidance

8. Need Help?

Shadow Systems includes basic setup support with premade bot purchases.

  • Join our Discord
  • Open a support ticket
  • Explain the issue clearly
  • Send screenshots if needed
  • Do not share your token publicly

Need Installation Help?

If you purchased a bot and need help installing it, join our Discord and open a support ticket. We will help you get started.
